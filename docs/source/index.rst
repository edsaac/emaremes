`emaremes`
========================

`emaremes` is a Python library to manage and speed up operations
with GRIB files from the `MRMS <https://www.nssl.noaa.gov/projects/mrms/>`_ project. 

.. image:: _static/test_plotting/CONUS_precip_flag.png
  :width: 100%
  :alt: Alternative text

These are convenient functions for

- downloading and organizing GRIB files locally
- building time series from points or polygons
- plotting maps 

.. note::
   This project is under active development.
