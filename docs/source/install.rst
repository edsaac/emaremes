Installation
============

To use emaremes, first install it using pip:

.. code-block:: console

   (.venv) $ pip install emaremes


Required dependencies
----------------------

- numpy
- matplotlib
- cartopy
- geopandas
- requests
- xarray
- cfgrib
- cmocean
- scipy

Optional dependencies
----------------------

*For testing:*

- pytest

*For documentation:*

- sphinx
- furo
- nbsphinx
- ipython
